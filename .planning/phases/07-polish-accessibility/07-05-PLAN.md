Edit `PhotoEditor/Editor/FilterStripView.swift`:

1. Selection ring color: replace `Color.accentColor` (in the `.stroke(...)` modifier on the cell overlay) with `Theme.Colors.accent`.
2. Wrap selection-ring change in animation:
   ```swift
   .overlay(
       RoundedRectangle(cornerRadius: 12)
           .stroke(isSelected ? Theme.Colors.accent : Color.clear, lineWidth: 3)
           .animation(Motion.adaptive(Motion.snappy), value: isSelected)
   )
   ```
3. Filter title text:
   ```swift
   Text(filter.displayName)
       .font(Theme.Typography.caption)
       .lineLimit(1)
       .foregroundStyle(isSelected ? Theme.Colors.text : Theme.Colors.secondary)
   ```
4. "Filters" header: `Text("Filters").font(Theme.Typography.subtitle)`
5. Strength section:
   - "Strength" label: `.font(Theme.Typography.subtitle)`
   - Percent text: `.font(Theme.Typography.valueBubble)` and `.foregroundStyle(Theme.Colors.secondary)`
   - Slider tint: replace `.tint(.blue)` with `.tint(Theme.Colors.accent)`
6. Add haptics:
   - In `.onTapGesture` (filter cell): play before selectFilter — but ONLY if the filter id changes:
     ```swift
     .onTapGesture {
         if selectedFilterID != filter.id { Haptic.play(.filterSelect) }
         viewModel.selectFilter(id: filter.id)
     }
     ```
   - In `.onLongPressGesture` (favorite toggle):
     ```swift
     .onLongPressGesture(minimumDuration: 0.4) {
         viewModel.filterLibrary.toggleFavorite(filter.id)
         Haptic.play(.recipeApply)   // success notification — favoriting is a success action
     }
     ```
7. Background placeholder for unloaded thumbnail: replace `Color(.tertiarySystemBackground)` with `Theme.Colors.panel`.
8. Preserve existing `.accessibilityLabel(...)` and `.accessibilityAddTraits(...)` calls (UX-05).
